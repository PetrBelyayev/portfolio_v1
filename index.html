Pokemon API

A single page website that uses a Pokemon API to fetch indvidual data about each pokemon. Cards can be flipped for additional information, clicking on another card or scrolling will close previous card. The page only fetches generation 1 pokemons.

Nasa API

Originally, a Shopify front end challenge for the Summer 2022 Internship, that ended up becoming my own personal project. This project uses NASA's APOD API which fetches pictures, date, and title for each item. I included a preloader animation while the page fetches the images, each card also includes a like/unlike button.
